# Doğrulama

ViraStack Mask, **Zod** ve **Yup** gibi popüler doğrulama kütüphanelerinin yanı sıra standart **React Hook Form** doğrulaması ile sorunsuz çalışacak şekilde tasarlanmıştır.

ViraStack Mask, dahili doğrulayıcı işlevlerini dışa aktarır, böylece bunları şemalarınızda doğrudan kullanabilirsiniz:

- `validateTCKN(value)`
- `validateLuhn(value)`
- `validateVKN(value)`
- `validateIBAN(value)`
- `validateEmail(value)`
- `validateURL(value)`
- `validateDate(value, format)`
- `validateExpiry(value)`

## Zod

ViraStack Mask'ın doğrulayıcılarını uygulamak için Zod'un `refine` yöntemini kullanın.

```tsx
import { useViraMask, validateTCKN, validateLuhn } from '@virastack/mask';
import { useForm } from 'react-hook-form';
import { zodResolver } from '@hookform/resolvers/zod';
import { z } from 'zod';

const schema = z.object({
  // TCKN Doğrulaması
  tckn: z.string()
    .length(11, "TCKN 11 haneli olmalıdır")
    .refine(validateTCKN, "Geçersiz TCKN"),

  // Kredi Kartı Doğrulaması (Luhn)
  creditCard: z.string()
    .min(1, "Kart numarası zorunludur")
    .refine(validateLuhn, "Geçersiz kart numarası"),
});

export default function ZodForm() {
  const form = useForm({
    resolver: zodResolver(schema),
    mode: 'onBlur'
  });

  const { tckn, creditCard } = useViraMask({
    form,
    schema: {
      tckn: 'tckn',      // Maske: 99999999999
      creditCard: 'card' // Maske: 9999 9999...
    }
  });

  return (
    <form onSubmit={form.handleSubmit(console.log)}>
      <div>
        <input {...tckn} placeholder="TCKN" />
        {form.formState.errors.tckn && (
          <span>{form.formState.errors.tckn.message}</span>
        )}
      </div>

      <div>
        <input {...creditCard} placeholder="Kart Numarası" />
        {form.formState.errors.creditCard && (
          <span>{form.formState.errors.creditCard.message}</span>
        )}
      </div>

      <button type="submit">Gönder</button>
    </form>
  );
}
```

## Yup

ViraStack Mask'ın doğrulayıcılarını uygulamak için Yup'un `test` yöntemini kullanın.

```tsx
import { useViraMask, validateTCKN } from '@virastack/mask';
import { useForm } from 'react-hook-form';
import { yupResolver } from '@hookform/resolvers/yup';
import * as yup from 'yup';

const schema = yup.object({
  tckn: yup.string()
    .length(11, "TCKN 11 haneli olmalıdır")
    .test('tckn', 'Geçersiz TCKN', validateTCKN)
    .required(),
});

export default function YupForm() {
  const form = useForm({
    resolver: yupResolver(schema)
  });

  const { tckn } = useViraMask({
    form,
    schema: { tckn: 'tckn' }
  });

  return (
    <form onSubmit={form.handleSubmit(console.log)}>
      <input {...tckn} />
      {form.formState.errors.tckn && (
        <span>{form.formState.errors.tckn.message}</span>
      )}
      <button type="submit">Gönder</button>
    </form>
  );
}
```

## React Hook Form (Standart)

Bir şema doğrulama kütüphanesi kullanmıyorsanız, React Hook Form'un yerleşik `validate` kuralını kullanabilirsiniz.

> **Not:** Bir çözümleyici (resolver) olmadan `useViraMask` kullanıldığında, ViraStack Mask varsayılan doğrulayıcılarını otomatik olarak `register` işlevine enjekte eder. Ancak, gerekirse bu davranışı manuel olarak geçersiz kılabilir veya genişletebilirsiniz.

```tsx
import { useViraMask, validateTCKN } from '@virastack/mask';
import { useForm } from 'react-hook-form';

export default function StandardForm() {
  const form = useForm();

  const { tckn } = useViraMask({
    form,
    schema: {
      tckn: {
        preset: 'tckn',
        // ViraStack Mask bunu otomatik olarak işler, ancak manuel olarak nasıl yapacağınız aşağıda açıklanmıştır:
        // validator: validateTCKN 
      }
    }
  });

  // Veya register aracılığıyla manuel olarak ekstra doğrulama uygulama (gelişmiş)
  /*
  <input 
    {...tckn}
    {...form.register('tckn', {
      validate: (value) => validateTCKN(value) || "Geçersiz TCKN"
    })}
  />
  */

  return (
    <form onSubmit={form.handleSubmit(console.log)}>
      <input {...tckn} />
      {form.formState.errors.tckn && (
        <span>Geçersiz TCKN</span>
      )}
      <button type="submit">Gönder</button>
    </form>
  );
}
```

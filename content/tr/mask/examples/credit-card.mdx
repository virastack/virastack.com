# Kredi Kartı

ViraStack Mask'ın `card` hazır ayarı American Express kartlarını otomatik olarak algılar. Numara **34** veya **37** ile başlıyorsa, maske 4-6-5 formatına (`9999 999999 99999`) geçer ve **CVV maskesi otomatik olarak 4 haneye geçer**. Aksi takdirde, standart 16 haneli formatı ve 3 haneli CVV'yi kullanır.

<CreditCardDemo />

```tsx
import { useViraMask } from '@virastack/mask';
import { useForm } from 'react-hook-form';

const form = useForm();
const { cardNumber, expiry, cvv } = useViraMask({
  form,
  schema: {
    cardNumber: 'card',
    expiry: 'expiry',
    cvv: 'cvv',
  },
});

// Kart stili düzeni
<form>
  <div>
    <label>Kredi Kartı Numarası</label>
    <input {...cardNumber} placeholder="4532 1234 5678 9012" />
  </div>
  <div>
    <div>
      <label>Son Kullanma Tarihi</label>
      <input {...expiry} placeholder="12/26" />
    </div>
    <div>
      <label>CVV</label>
      <input {...cvv} placeholder="123" />
    </div>
  </div>
</form>
```

# Şifre

`password` hazır ayarı herhangi bir karakter girişine izin verir. Görsel maskelemenin giriş türü tarafından işlendiği şifre alanları için uygundur.

<MaskedInputDemo preset="password" label="Şifre" />

```tsx
import { useViraMask } from '@virastack/mask';
import { useForm } from 'react-hook-form';

const form = useForm({ defaultValues: { password: '' } });
const { password } = useViraMask({
  form,
  schema: { password: 'password' },
});

<input {...password} />
```

# Posta Kodu

`zipCode` hazır ayarı posta kodlarını biçimlendirir. Girişi standart posta kodu formatıyla sınırlar.

<MaskedInputDemo preset="zipCode" label="Posta Kodu" />

```tsx
import { useViraMask } from '@virastack/mask';
import { useForm } from 'react-hook-form';

const form = useForm({ defaultValues: { zipCode: '' } });
const { zipCode } = useViraMask({
  form,
  schema: { zipCode: 'zipCode' },
});

<input {...zipCode} />
```

# URL

`url` hazır ayarı URL girişlerini biçimlendirir ve doğrular. Girişin geçerli bir URL yapısını izlemesini sağlar.

<MaskedInputDemo preset="url" label="Web Sitesi URL'si" />

```tsx
import { useViraMask } from '@virastack/mask';
import { useForm } from 'react-hook-form';

const form = useForm({ defaultValues: { url: '' } });
const { url } = useViraMask({
  form,
  schema: { url: 'url' },
});

<input {...url} />
```

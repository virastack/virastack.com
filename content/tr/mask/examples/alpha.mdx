# Alfabetik

`alpha` hazır ayarı girişi yalnızca alfabetik karakterlerle sınırlar. İsimler veya diğer yalnızca metin girişleri gibi alanlar için kullanışlıdır.

<MaskedInputDemo preset="alpha" labelKey="firstName" />

```tsx
import { useViraMask } from '@virastack/mask';
import { useForm } from 'react-hook-form';

const form = useForm({ defaultValues: { name: '' } });
const { name } = useViraMask({
  form,
  schema: { name: 'alpha' },
});

<input {...name} />
```

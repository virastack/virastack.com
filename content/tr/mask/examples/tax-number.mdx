# Vergi Numarası

`taxNumber` hazır ayarı vergi kimlik numaralarını (VKN) biçimlendirir. Vergi numarasının uzunluğunu ve yapısını doğrular.

<MaskedInputDemo preset="taxNumber" label="Vergi Kimlik Numarası" />

```tsx
import { useViraMask } from '@virastack/mask';
import { useForm } from 'react-hook-form';

const form = useForm({ defaultValues: { taxNumber: '' } });
const { taxNumber } = useViraMask({
  form,
  schema: { taxNumber: 'taxNumber' },
});

<input {...taxNumber} />
```

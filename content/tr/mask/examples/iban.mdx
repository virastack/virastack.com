# IBAN

`iban` hazır ayarı Uluslararası Banka Hesap Numaralarını biçimlendirir. Daha iyi okunabilirlik ve doğrulama için karakterleri gruplandırır.

<MaskedInputDemo preset="iban" label="IBAN" />

```tsx
import { useViraMask } from '@virastack/mask';
import { useForm } from 'react-hook-form';

const form = useForm({ defaultValues: { iban: '' } });
const { iban } = useViraMask({
  form,
  schema: { iban: 'iban' },
});

<input {...iban} />
```

# Sayısal

`numeric` hazır ayarı girişi yalnızca sayısal karakterlerle sınırlar. Sayımlar, miktarlar veya diğer tamsayı değerleri gibi alanlar için uygundur.

<MaskedInputDemo preset="numeric" label="Miktar" />

```tsx
import { useViraMask } from '@virastack/mask';
import { useForm } from 'react-hook-form';

const form = useForm({ defaultValues: { count: '' } });
const { count } = useViraMask({
  form,
  schema: { count: 'numeric' },
});

<input {...count} />
```

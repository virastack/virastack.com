# Telefon

`phone` hazır ayarı girişi bir telefon numarası olarak biçimlendirir. Varsayılan maske, daha iyi okunabilirlik için **Türkiye** formatını izler (örn. `(555) 555 55 55`).

<MaskedInputDemo
  preset="phone"
  label="Telefon Numarası (TR)"
  placeholder="(555) 555 55 55"
/>

```tsx
import { useViraMask } from '@virastack/mask';
import { useForm } from 'react-hook-form';

const form = useForm();
const { phone } = useViraMask({
  form,
  schema: { phone: 'phone' },
});

<input {...phone} placeholder="(555) 555 55 55" />
```

## Özelleştirme

Diğer bölgeleri veya formatları desteklemek için varsayılan maskeyi geçersiz kılabilirsiniz. Aşağıdaki örnek ABD tarzı bir maske kullanır:

<MaskedInputDemo
  preset="phone"
  label="Telefon Numarası (ABD)"
  placeholder="(555) 555-5555"
  options={{
    mask: '(999) 999-9999',
  }}
/>

```tsx
import { useViraMask } from '@virastack/mask';
import { useForm } from 'react-hook-form';

const form = useForm();
const { phone } = useViraMask({
  form,
  schema: {
    phone: {
      mask: '(999) 999-9999',
    },
  },
});

<input {...phone} placeholder="(555) 555-5555" />
```

Daha fazla ayrıntı için [Maske Seçenekleri](/mask/mask-options) sayfasına bakın.

# Metin

`text` hazır ayarı, belirli bir biçimlendirme olmadan herhangi bir metin girişine izin verir. Standart bir metin giriş alanı olarak işlev görür.

<MaskedInputDemo preset="text" label="Ad Soyad" />

```tsx
import { useViraMask } from '@virastack/mask';
import { useForm } from 'react-hook-form';

const form = useForm({ defaultValues: { text: '' } });
const { text } = useViraMask({
  form,
  schema: { text: 'text' },
});

<input {...text} />
```

# Son Kullanma Tarihi

`expiry` hazır ayarı girişi bir son kullanma tarihi (AA/YY) olarak biçimlendirir. Yaygın olarak kredi kartı son kullanma alanları için kullanılır.

<MaskedInputDemo preset="expiry" label="Son Kullanma Tarihi (AA/YY)" />

```tsx
import { useViraMask } from '@virastack/mask';
import { useForm } from 'react-hook-form';

const form = useForm({ defaultValues: { expiry: '' } });
const { expiry } = useViraMask({
  form,
  schema: { expiry: 'expiry' },
});

<input {...expiry} />
```

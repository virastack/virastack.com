# E-posta

`email` hazır ayarı, giriş biçimini doğrulayarak geçerli bir e-posta adresine benzemesini sağlar. E-posta alanlarında yaygın yazım hatalarını önlemeye yardımcı olur.

<MaskedInputDemo preset="email" labelKey="emailAddress" />

```tsx
import { useViraMask } from '@virastack/mask';
import { useForm } from 'react-hook-form';

const form = useForm({ defaultValues: { email: '' } });
const { email } = useViraMask({
  form,
  schema: { email: 'email' },
});

<input {...email} />
```

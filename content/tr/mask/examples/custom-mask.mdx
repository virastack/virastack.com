# Özel Maske

Belirli ihtiyaçlar için özel maskeler ve doğrulayıcılar oluşturabilirsiniz.

## Çift Sayı Doğrulayıcı

Bu örnek, yalnızca sayıları kabul eden ve sayının çift olup olmadığını doğrulayan özel bir maskeyi gösterir.

<CustomMaskDemo />

```tsx
import { useViraMask } from '@virastack/mask';
import { useForm } from 'react-hook-form';

const form = useForm();
const { formState: { errors } } = form;
const { evenNumber } = useViraMask({
  form,
  schema: {
    evenNumber: {
      inputMode: 'numeric',
      allowedChars: /[0-9]/,
      validate: true,
      validator: (value) => parseInt(value, 10) % 2 === 0,
    },
  },
});

<input {...evenNumber} />
{errors.evenNumber && <span>Çift sayı olmalıdır</span>}
```

Özelleştirme hakkında daha fazla ayrıntı için [Maske Seçenekleri](/mask/mask-options) sayfasına bakın.

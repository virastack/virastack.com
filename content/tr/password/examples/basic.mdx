# Temel Kullanım

Bu örnek, `useViraPassword` hook'unun standart HTML `input` ve `button` öğeleriyle nasıl kullanılacağını gösterir.

<BasicDemo />

```tsx
import { useViraPassword } from '@virastack/password';

export function BasicDemo() {
  const { inputProps, btnProps } = useViraPassword();

  return (
    <div>
      <input {...inputProps} />
      <button {...btnProps} />
    </div>
  );
}
```

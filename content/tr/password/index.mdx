## Hızlı Başlangıç

```tsx
import { useViraPassword } from '@virastack/password';

function PasswordInput() {
  const { inputProps, btnProps } = useViraPassword();

  return (
    <div>
      <input {...inputProps} />
      <button {...btnProps}>
        {btnProps.children}
      </button>
    </div>
  );
}
```

## Özellikler

- 🔒 **Güvenli**: Şifre görünürlüğünü güvenli bir şekilde yönetir.
- ♿ **Erişilebilir**: Ekran okuyucular için tam ARIA desteği.
- 🎨 **Özelleştirilebilir**: Değiştirme butonu için özel ikonlar veya metin kullanın.
- 📦 **Hafif**: Sıfır bağımlılık (React hariç).
- 🧩 **Kolay Entegrasyon**: Herhangi bir UI kütüphanesi (shadcn/ui, MUI vb.) ile sorunsuz çalışır.

# Başlarken

## Kurulum

Paketi tercih ettiğiniz paket yöneticisini kullanarak yükleyin:

<CodeBlockCommand {...convertNpmCommand("npm install @virastack/password")} />

## Temel Kullanım

`useViraPassword` hook'unu içe aktarın ve girdi alanınızı ve butonunuzu bağlamak için döndürülen propları kullanın.

```tsx
import { useViraPassword } from '@virastack/password';

function PasswordInput() {
  const { inputProps, btnProps } = useViraPassword();

  return (
    <div>
      <input {...inputProps} />
      <button {...btnProps}>
        {btnProps.children}
      </button>
    </div>
  );
}
```

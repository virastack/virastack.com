# Zip Code

The `zipCode` preset formats postal codes. It restricts the input to the standard zip code format.

<MaskedInputDemo preset="zipCode" label="Zip Code" />

```tsx
import { useViraMask } from '@virastack/mask';
import { useForm } from 'react-hook-form';

const form = useForm({ defaultValues: { zipCode: '' } });
const { zipCode } = useViraMask({
  form,
  schema: { zipCode: 'zipCode' },
});

<input {...zipCode} />
```

# Expiry

The `expiry` preset formats the input as an expiration date (MM/YY). It is commonly used for credit card expiration fields.

<MaskedInputDemo preset="expiry" label="Expiration Date (MM/YY)" />

```tsx
import { useViraMask } from '@virastack/mask';
import { useForm } from 'react-hook-form';

const form = useForm({ defaultValues: { expiry: '' } });
const { expiry } = useViraMask({
  form,
  schema: { expiry: 'expiry' },
});

<input {...expiry} />
```

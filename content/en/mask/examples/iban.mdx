# IBAN

The `iban` preset formats International Bank Account Numbers. It groups the characters for better readability and validation.

<MaskedInputDemo preset="iban" label="IBAN" />

```tsx
import { useViraMask } from '@virastack/mask';
import { useForm } from 'react-hook-form';

const form = useForm({ defaultValues: { iban: '' } });
const { iban } = useViraMask({
  form,
  schema: { iban: 'iban' },
});

<input {...iban} />
```

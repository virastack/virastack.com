# Examples

Explore the various ways to use the `useViraPassword` hook in your React applications.

- **[Self-Closing & Positioning](/password/examples/self-closing)**: Using self-closing button syntax and absolute positioning.
- **[Custom Icons](/password/examples/custom-icons)**: Using custom icons for the toggle button.
- **[Custom Text](/password/examples/custom-text)**: Using text labels instead of icons.
- **[Shadcn UI Integration](/password/examples/shadcn)**: Seamless integration with Shadcn UI components.
